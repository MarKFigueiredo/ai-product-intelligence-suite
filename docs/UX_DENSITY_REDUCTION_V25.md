# UX Density Reduction — v25

v25 keeps deep evidence available but makes the first portfolio experience lighter.

## Changes

- Guided Demo remains the default workflow in Focused review mode.
- The hero workflow is now an 8-step path instead of a broader 10-step list.
- The UI shows a progress control: `Step 1/8` through `Step 8/8`.
- The default view shows the current step first.
- The full step list is moved into an expander.
- Evidence, lineage, gate details and tables stay available on demand.

## Design rule

```text
Default UI = compact decision path.
Expanders/docs = full evidence and deep review.
```

This keeps the project reviewable by recruiters and hiring managers without hiding the technical depth from Principal PM, QA, Compliance or Engineering reviewers.
