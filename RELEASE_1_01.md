# Release 1.01 — First Public Portfolio Release

This is the first version intended for public GitHub/portfolio review.

It is based on the internal v25 work, but the public version is labelled **1.01** to avoid exposing a long private iteration history as the headline release.

## What this release demonstrates

- Core compliance-to-product pipeline.
- Human feedback flywheel.
- Mandatory negative test coverage.
- Citation verifier and validation pack.
- Swiss QR-Bill second-domain validation case.
- Claim hygiene scanner.
- Focused portfolio review path with full artifact map available.

## What this release does not claim

- It is not production SaaS.
- It does not guarantee compliance.
- It does not replace legal, compliance, QA or product review.
- Synthetic benchmark examples are separated from real local usage metrics.
