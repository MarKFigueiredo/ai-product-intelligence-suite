# AI Payments Compliance Assistant

This folder contains a working playbook draft for a separate MIT Sloan course project.

It is included as a **companion artifact**, not as a new Streamlit app module.

## Why it is included

The main portfolio app shows how to build an AI-assisted compliance-to-product workflow in Python/Streamlit. This companion playbook shows how the same product thinking can be applied to a different domain and implementation style: payments compliance and agentic workflow adoption.

## Course context

Course: Implementing Agentic AI: Building Your Organizational Playbook  
Use case: SEPA / ISO 20022 structured-address changes  
Status: working draft to be refined with the course frameworks

## Intended future project

A separate low-code/agentic workflow using tools such as n8n, Airtable/Sheets, LLM APIs and Slack/email handoff.

## Boundary

This is not production software and does not make legal or compliance determinations. It is a course/playbook artifact for thinking through workflow design, governance, adoption and risk.
